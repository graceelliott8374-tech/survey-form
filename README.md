# Survey Form
This was my first web development project, completed during the PERSEVERE Coding Bootcamp while I was incarcerated.
The project is preserved in its original form to document the beginning of my development journey. It contains mistakes I would approach differently today, including duplicate IDs, invalid HTML structure, heavy use of inline CSS, and accessibility issues.

## Technologies Used
-HTML
-Inline CSS

## What I Learned
This project introduced me to:
-HTML page structure
-Forms and user input
-Labels and form controls
-Required fields and validation
-Basic CSS styling
-Organizing content on a webpage

## Project Status
Completed and preserved as originally written

## Background
Rather than rewriting the code, I chose to keep the original version intact. My portfolio website will explain the mistakes in this project, what I learned from them, and how my development practices improved over time.
